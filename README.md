# Persistent Vehicle HP Bars

## 📜 Description

Persistent Vehicle HP Bars is a mod for `Company of Heroes Relaunch` built on top of `CoHModSDK` which
always displays the HP bars for vehicles.

## 🔧 Installation

1. Install the `CoHModSDK` loader in your game directory
2. Copy `PersistentVehicleHPBars.dll` into the `mods` folder
3. Add `PersistentVehicleHPBars.dll` to `CoHModSDKLoader.ini`
4. Start the game

## 🛠️ Building

If you want to build the mod yourself, open the solution in Visual Studio 2022 and build `Release|x86`.
